## Design System

### Colors
```css
Primary: #1E3A8A (Navy Blue)
Success: #10B981 (Green)
Warning: #F59E0B (Amber)
Error: #EF4444 (Red)
Info: #3B82F6 (Blue)
Purple: #8B5CF6 (Accent)
```

### Typography
```css
Headings: Poppins (700)
Body: Inter (400-600)
Monospace: System mono
```

### Spacing
```css
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
```
